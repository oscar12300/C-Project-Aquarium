# 開心水族箱
### 1.結構 編號1~5號魚
- 編號 int
- 名稱	char name[20]
- 顏色 char
- 體型 float 5-15cm	每餵食兩次+0.5
- 心情 int 1-5 (難過~開心)	互動可+1,每做三次動作使其他魚的心情-1
- 飢餓度 int 0-5 初始值為3，等於0時生命-1，每做四次動作-1
- 生命 int 0-5	為0時死亡 每餵食兩次+0.5，飢餓為0或心情為1時-1
